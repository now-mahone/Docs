# Kerne Protocol

**Primary Objective:** Make the owners as much money as possible, as quickly as possible, and as easily as possible.

Kerne is a delta-neutral synthetic dollar protocol on the Base network, leveraging LST collateral and CEX-based hedging to achieve its primary objective of maximizing owner wealth.

## Core Premise
The protocol is engineered for maximum capital efficiency and institutional-grade security. All architectural decisions, features, and strategies are evaluated based on their contribution to the protocol's long-term stability and growth.

## Technical Stack
- **Smart Contracts:** Solidity 0.8.24, Foundry, OpenZeppelin v5.0.
- **Hedging Engine:** Python 3.10, CCXT, Web3.py, Docker.
- **Frontend:** Next.js 15, Tailwind CSS, Wagmi/Viem.
